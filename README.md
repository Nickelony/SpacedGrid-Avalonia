# AvaloniaSpacedGrid
An extension of AvaloniaUI's Grid class with added RowSpacing and ColumnSpacing properties.

NuGet: https://www.nuget.org/packages/AvaloniaSpacedGrid/

WPF version: https://github.com/Nickelony/SpacedGrid-WPF
